You can use the regrid module to map scattered data onto a grid, which
should allow you to make contour plots, use rubber grid etc.

see http://opendx.informatics.jax.org/docs/html/pages/refgu115.htm#HDRREGRID

I have played with this before, however, my data sets have always been
to big for this to be feasible for me.
