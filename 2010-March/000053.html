It's hard to read the documentation at the very beginning.
In any case, something like this should work:

1) use "construct" to build a regular x,y,z mesh (qith only one z value)
2) "mark" the position of the resulting field
3) use "compute" on the marked filed in order to compute z
4) "replace" the "data" of point 1) with the result of 3)
5) "rubbersheet"
6) "image"

or

3) "compute" a vector with your desired z:
    for example, [a.0, a.1, sin(a.0)*sin(a.1)]
4) "replace" the "positions" of point 1) with the "data"
    field of point 3)
5) "show connections"
6) "image"

then you can add contour, using position.z as data ...
