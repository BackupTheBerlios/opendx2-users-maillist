> Anyone out there?   I'm wondering if anyone is interested in some work I've been doing on OpenDX - I've replaced the multi-process model with a multi-threaded one, enabling a much (much!) simplified memory manager.  This enables me to remove the limit on memory size (though I bet you'd better not allocate any individual object larger than 2GB).  It'll just consume virtual-memory page-space until it hits an arbitrary limit (suggestions?) when it will begin to flush the cache.  Fragmentation is also much less of a problem.  
>
